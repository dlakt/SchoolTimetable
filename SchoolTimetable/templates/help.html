📖 Помощь по программе

🎯 Назначение программы

Программа предназначена для автоматического составления школьного расписания с учетом всех ограничений.

⚙️ Как пользоваться

  1. На главной странице выберите класс, для которого нужно составить расписание
  2. Нажмите на кнопку с названием класса
  3. Программа автоматически сгенерирует оптимальное расписание
  4. Вы можете просмотреть расписание, скопировать его или распечатать
  5. Для генерации расписания для всех классов используйте кнопку "Сгенерировать для всех классов"

📊 Оценка качества расписания

  • 70-100% - Отличное расписание, минимум "окон"
  • 40-69% - Хорошее расписание, есть небольшие недочеты
  • 0-39% - Требуется доработка, много "окон"

📁 Поддерживаемые данные

Программа использует файл data/school_data.json со следующей структурой:

  • teachers - список преподавателей
  • classes - список классов
  • classrooms - список кабинетов
  • subjects - список предметов с количеством часов
  • settings - настройки расписания

🔧 Алгоритм работы

Программа использует жадный алгоритм с эвристиками:

  1. Сложные предметы (математика, физика, языки) ставятся в первой половине дня
  2. Учитывается занятость учителей (один учитель не может вести два урока одновременно)
  3. Учитывается специализация кабинетов (физика, химия, информатика, спортзал)
  4. Минимизируется количество "окон" в расписании учеников

⚠️ Возможные ошибки

  • "Не удалось разместить предмет" - недостаточно свободных слотов для предмета
  • "Класс не найден" - проверьте корректность данных в school_data.json
